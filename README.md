# ✦ Between · 人生可视化

> 人生被切割成一个个节点——毕业、工作、买房、稳定  
> 好似只有等到下一个节点到来，才有资格做想做的事  
> 但打卡只是瞬间，**点与点之间，才是生活。**

**Between** 是一个把一个人的一生画成星座的网站。  
从宇宙中最亮的一颗星出发，随着镜头拉近，逐渐展开为生与死、各个人生阶段、乃至每一条当下的记录。  
每一次记录，都是一颗亮起来的星——让原本空洞的人生线，慢慢变成只属于你自己的星座。

## 💡 核心理念

大多数记录类产品只在堆叠碎片。**Interlude** 想回答的是一个更本质的问题——

> 我的一生，长什么样子？  
> 我有没有真的在好好地过自己的生活？

通过可视化的方式，让「活在当下」从一句口号，变成一个可以观测、可以积累的数字体验：

- 如果没有记录，人生只是孤零零的几个节点，甚至不是一条连续的线
- 每加入一条记录，那段星线就会变粗一点、变亮一点
- 哪个阶段的星线最厚，哪个阶段你就活得最用心

## 🌌 功能一览

### 宏观视角 · 人生星座

| 功能 | 说明 |
|---|---|
| 多尺度缩放 | 从「宇宙一颗星」逐级拉近——宇宙视角 → 生死两点 → 人生星座，由虚走向实 |
| 星线密度可视化 | 记忆越多，星线越粗越亮，人生厚度直接可见（5 个密度等级）|
| 拖拽星点移位 | 每颗节点星可自由拖拽，重新布局你的星图，保存后持久保持 |
| 节点调色盘 | 每颗星独立配色，16 个预设低饱和度色调 + 原生自定义取色器 |
| 编辑主线 | 自由增删节点，支持两种模式：**人生节点**（自定义里程碑）/ **时间节点**（年份刻度）|
| 微观宏观联动 | 时间节点模式下，微观记录按日期自动归入宏观对应年份节点 |

### 微观视角 · 每月记录

| 功能 | 说明 |
|---|---|
| 每月时间线 | 横向时间线展示一整月；有记录的日期直接变亮，无记录的日期呈暗淡虚影 |
| S 形日历视图 | 缩小后时间线弯曲成蛇形日历，7 天一行，一屏看完整个月 |
| 朋友圈式卡片 | 放大后每条记录从竖线茎上延伸出卡片，含文字 + 多图 + 时间 |
| 电子云小星星 | 缩小时记录自动收缩为漂浮的小星点（防重叠），点击仍可查看 |
| 拖拽上传图片 | 直接把图片拖入上传区，最多 4 张，横竖图自动识别等比显示 |
| 留言功能 | 每条记录右下角 💬 图标，点击展开留言框，Enter 快捷发送 |

### 通用

- **零账号、零成本**：所有数据存储在浏览器本地（`localStorage`），不上传、不联网、不依赖任何服务器
- **「继续上次」**：主页保留上次的状态，刷新不丢失
- **移动端支持**：支持双指缩放、单指平移，手机浏览器可正常使用

## 🚀 快速开始

### 方式一：本地直接运行（推荐体验）

```bash
git clone https://github.com/your-username/interlude.git
```

然后直接双击 `index.html` 在浏览器中打开即可。无需安装任何依赖。

### 方式二：部署到 GitHub Pages（免费公开演示）

1. Fork 或 clone 本仓库到你的 GitHub 账号
2. 进入仓库 → **Settings → Pages**
3. Source 选择 `main` 分支，目录选 `/` (root)，点击 Save
4. 等约 1 分钟，访问 `https://your-username.github.io/interlude`

### 方式三：其他静态平台

由于是纯静态文件，任何可以托管 HTML 的平台均可部署：

- [Vercel](https://vercel.com) — 拖入文件夹即可
- [Netlify](https://netlify.com) — 同上


没有框架，没有构建工具，没有 npm。一个 `index.html` 就是全部。
